# 文本挖掘准确率评估模板

## 1. 建议采用的最小可交方案

不必做复杂 benchmark，可以采用人工抽样评估。

推荐流程：

1. 从抽取结果中随机抽取 30 到 50 篇摘要。
2. 对每篇摘要记录：
   - 实体识别是否正确
   - 关系抽取是否合理
3. 统计正确数、错误数和典型误差类型。
4. 在报告中给出简述和小表格即可。

## 3. 报告中可直接使用的写法模板

本项目采用人工抽样的方式对文本挖掘结果进行简要评估。我们从自动抽取结果中随机抽取若干篇与 LINE-1 相关的文献摘要，对实体识别结果和关系抽取结果进行人工核查。评估重点包括：转座元件名称识别是否准确、疾病与功能实体识别是否合理、关系方向与语义是否符合原文描述。

## 5. 误差分析可写方向

- 同一实体存在多种命名形式，导致别名未完全统一。
- 某些疾病和功能短语边界不稳定。
- 某些抽取结果会把机制词误识别为疾病，或把疾病误识别为功能。
- 某些关系词语义过宽，需要后续在图数据库导入前做规范化。
