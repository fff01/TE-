# 目录文件说明

本文档用于快速说明 `D:\wamp64\www\TE-` 目录下各文件与子目录的内容和作用，便于项目交接、答辩前检查与后续维护。

说明原则：
- `.git/` 为 Git 仓库内部元数据目录，包含提交历史、对象库、引用、日志等标准版本控制文件，不逐个展开。
- `__pycache__/` 为 Python 运行缓存目录，不属于源代码主体。
- 以下以“目录/文件 -> 作用”的方式整理，优先覆盖当前项目中实际有用的文件。
