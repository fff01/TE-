# reference 目录说明

这个目录用于集中存放**不参与当前网站运行主链**的参考资料、历史归档与展示材料。

当前包含：

- `g6-official/`
  - 本地 G6 官方源码、示例、文档参考区
- `archive/`
  - 历史脚本与归档页面
- `latex-report/`
  - 报告写作与图文材料
- `presentation-materials/`
  - 展示稿、答辩辅助说明

## 当前原则

- 这些内容默认不作为网站运行依赖。
- 当前网站运行主线仍然在项目根目录的：
  - `api/`
  - `assets/`
  - `data/`
  - 顶层 PHP / HTML 页面
- 若后续某份参考资料重新进入开发主链，应先明确依赖关系，再决定是否迁回主工作区。
