# 实验报告材料准备清单

## 一、正式写作前必须准备的材料

### 1. 项目概述材料
- 项目题目
- 课程名称
- 小组成员姓名与学号
- GitHub 仓库地址
- 开发环境与技术栈

### 2. 文献挖掘与信息抽取材料
- PubMed 检索关键词
- 抓取字段列表：PMID、Title、Abstract、Year、Journal
- 文献抓取脚本名称
- 信息抽取脚本名称
- 实体类型列表
- 关系类型列表
- 典型抽取示例 2 到 3 条

### 3. 数据清洗与规范化材料
- 去重策略
- 别名统一策略
- LINE-1 亚家族补关系说明
- 原始数据文件与清洗后文件名称

### 4. 准确率评估材料
- 人工抽样篇数
- 抽样方式
- 实体识别正确数
- 关系抽取正确数
- 误差类型总结

### 5. 图数据库 Schema 材料
- 节点类型表
- 关系类型表
- 关键属性表
- 设计原因说明

### 6. 数据导入与查询材料
- 关键导入脚本列表
- 关键 Cypher 查询列表
- 代表性查询结果截图

### 7. 可视化模块材料
- 页面结构说明
- 交互功能说明
- 图谱页面截图
- 节点详情截图
- 关系详情截图

### 8. 智能问答模块材料
- RAG 架构图
- Prompt 设计思路
- 问答接口流程
- 典型问答截图

### 9. 小组分工材料
- 每位成员负责的模块
- 每位成员的主要代码或数据工作

### 10. 答辩材料
- PPT 大纲
- 现场演示流程
- 备用演示问题

## 二、当前最优先补齐的 4 项
- 准确率评估结果
- RAG 架构图
- 网站截图
- 成员分工说明
