# 术语表说明

这个目录用于维护项目的中英术语映射，目标是把原本散落在前端代码中的：

- 中英文显示名
- 大小写变体
- 常见别名

统一收敛为可维护的数据文件。

## 文件

- `te_terminology.csv`
  - 适合人工维护
  - 后续新增数据时优先改这个文件
- `te_terminology.json`
  - 结构化版本
  - 便于后续前端 / 后端直接读取

## 字段说明

- `type`
  - 节点类型，如 `Disease`、`Function`、`Paper`
- `canonical_en`
  - 建议作为英文标准名使用的字段
- `display_zh`
  - 中文展示名
- `aliases_en`
  - 英文别名，使用 `|` 分隔
- `aliases_zh`
  - 中文别名，使用 `|` 分隔
- `notes`
  - 补充说明

## 建议维护规则

1. 后续数据库若仍以英文节点为主，优先把英文标准名填入 `canonical_en`
2. 前端展示统一使用 `display_zh`
3. 若发现新的英文变体，只补到 `aliases_en`
4. 若发现新的中文叫法，只补到 `aliases_zh`
5. 不要把明显不是同一个概念的项强行合并
